## Project Overview
This repository contains my Digital Literacy Project for the course CSE0001. The main aim of this project is to understand how to use digital tools properly, communicate professionally, and stay safe online.

Through this project, I worked on different tasks like creating an infographic, building my online profiles, exploring useful platforms, writing professional emails, and learning about cybercrime.

---

## Task Summary

### Task 1 - Infographic
I created a digital literacy infographic using Canva. It explains important topics like safe internet usage, useful tools, and maintaining a professional online presence.

### Task 2 - Digital Portfolio
I created and updated my profiles on GitHub, LinkedIn, and HackerRank. These platforms will help me in building my skills and professional presence.

### Task 3 - Platforms
I solved a beginner-level problem on HackerRank and created a Google Form quiz related to digital literacy.

Google Form Link: https://forms.gle/SUZYRhBL63pcCw5K8

### Task 4 - Email Etiquette
I wrote two professional emails, one for requesting an assignment extension and another for applying for an internship. I also created a checklist for responsible social media usage.

### Task 5 - Cybercrime Awareness
I worked on a case study based on UPI fraud and created a prevention checklist to spread awareness about online safety.
